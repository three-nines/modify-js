    Function wrapTrySync

    • wrapTrySync<T, E, R>(
          logic: () => T,
          cb_err?: (err: E) => R,
          cb_last?: (result: T | undefined) => void,
          isThrow?: boolean,
      ): T | R | undefined

      Synchronous try-catch wrapper for clean flow control.

      Type Parameters

      • T

        The return type of the main synchronous logic function.

      • E

        The expected error type caught in the catch block. Defaults to unknown.

      • R

        The return type of the error callback function.

      Parameters

      • logic: () => T

        The core synchronous function to execute. Must satisfy isFunction check.

      • Optionalcb_err: (err: E) => R

        Optional synchronous error callback.

      • Optionalcb_last: (result: T | undefined) => void

        Optional synchronous finalizer callback. Receives the successful result T, or undefined if execution failed.

      • OptionalisThrow: boolean

        If true, re-throws the caught error after executing callbacks.

      Returns T | R | undefined

      The result of logic on success, the result of cb_err on failure, or undefined if a failure occurs and no error callback is provided.

      Remarks

      CRITICAL: This utility executes purely synchronously. All provided callbacks (cb_err, cb_last) must be synchronous routines. If asynchronous functions or Promise-returning actions are provided, they will fire background tasks, but the wrapper will return immediately without waiting for them to complete.

      Throws

      If the provided logic argument fails the isFunction validation check.

      Throws

      Re-throws the original caught error if isThrow is true and an exception occurs during execution.

      Example

      const result = wrapTrySync(
          () => JSON.parse(configString),
          (err) => ({ fallback: true }),
          () => console.log("Sync parsing attempted"),
          false
      );
